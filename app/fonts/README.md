# Thai Fonts for PDF Generation

Place these files here:
- `Sarabun-Regular.ttf`
- `Sarabun-Bold.ttf`

## Download

1. Go to https://fonts.google.com/specimen/Sarabun
2. Click "Download family" (top right)
3. Extract the ZIP
4. Copy these files to this folder:
   - `static/Sarabun-Regular.ttf` → `app/fonts/Sarabun-Regular.ttf`
   - `static/Sarabun-Bold.ttf` → `app/fonts/Sarabun-Bold.ttf`
